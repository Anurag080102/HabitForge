# HabitForge

Android habit tracking app with reflective journaling and community milestone sharing.

## Team
- Anurag (Backend: Data models, Room, Firebase, API)
- Kelyan (Frontend: Jetpack Compose UI, Navigation, Animations)

## Features
- Offline-first habit tracking with streaks
- Daily journal entries
- Motivational quotes (external API)
- Community feed for shared milestones (Firebase sync)
- Multi-language support (EN/FR + Hindi)
